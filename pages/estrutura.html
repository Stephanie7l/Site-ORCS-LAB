Infraestrutura

Nossa Estrutura

Ambiente técnico e colaborativo para pesquisa aplicada em sistemas complexos.

Espaços Físicos

O laboratório ocupa uma área de aproximadamente 100 m², dividida em áreas específicas para estudo, pesquisa e desenvolvimento. Conta com mesas para trabalho colaborativo, trabalho individual e projetor para apresentações.

Capacidade

A estrutura comporta até 20 estudantes, oferecendo suporte para o desenvolvimento de projetos e colaborações.

Equipamentos

  • 13 estações de trabalho para uso individual.
  • 4 servidores locais para simulações e armazenamento de dados.

Softwares

  • MATLAB e Simulink.
  • Python com bibliotecas como NumPy, TensorFlow e Scikit-learn.
  • Plataformas de desenvolvimento colaborativo, como GitHub.

Recursos de Biblioteca

O laboratório conta com uma coleção de mais de 50 livros, abrangendo temas como otimização, inteligência artificial, ciência de dados, engenharia de sistemas e desenvolvimento de software.

Outros Recursos

Ambiente para longas jornadas de trabalho, cozinha equipada, área de convivência e rede Wi-Fi de alta velocidade.
